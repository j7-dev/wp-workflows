# Workflows（GitHub Copilot Coding Agent 專屬）

此目錄中的 `.md` 工作流程使用 **GitHub Copilot coding agent DSL** 格式定義，專為 GitHub Copilot coding agent 設計。

## 注意事項

- **Claude Code Plugin 不使用這些工作流程**。Claude Code 使用 `agents/` 和 `skills/` 目錄中的元件。
- 這些檔案保留作為參考資料，供需要在 GitHub Copilot coding agent 環境中使用的場景參考。
- 如需將工作流程邏輯遷移到 Claude Code，請參考 `agents/` 目錄中的 Agent 定義。
